## About The Project

This repository contains two seperate segements of Python code. 

WeatherPy is a Python script that randomly generates a list of over 500 cities and then uses the OpenWeatherMap API to create a representative weather model accross th worlds cities.
The results of the script are exported as both a raw data CSV and a PNG image of the created scatter plot.

VactaionPy is a Python script that uses the Google Places API and Jupyter Notebooks to find hotels in cities that meet certain weather requirements.
* Temp between 70 and 80 degrees fahrenheit
* Wind Speed less than 10mph
* No clouds
